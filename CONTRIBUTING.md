# Contributing

Thanks for your interest in improving this skill. Here's how to contribute.

## What we're looking for

- **Better prompt patterns** -- if you've found a structure that gets better results from GPT Pro, share it.
- **Domain-specific templates** -- specialized prompt templates for security audits, data analysis, writing review, etc.
- **Bug fixes** -- if the skill misses a sanitization case, breaks on certain file types, or has a logic error.
- **Documentation improvements** -- clearer explanations, better examples, typo fixes.

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b improve-security-template`)
3. Make your changes in `SKILL.md` or add templates
4. If possible, include before/after examples showing the improvement
5. Open a PR with a clear description of what changed and why

## Guidelines

- Keep prompts concise. Longer prompts are not better prompts.
- Test your changes by running the skill in Claude Code and uploading the result to ChatGPT Pro.
- Don't include real credentials, API keys, or sensitive data in examples.

## Questions?

Open an issue. We'll respond quickly.
